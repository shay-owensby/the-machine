# Employee Exit Checklist

A comprehensive checklist for closing out a departure completely — used after the separation type/risk branch is set (voluntary vs. involuntary vs. layoff; see the hr-offboarding skill). Adapt to the type: **involuntary and layoff paths require counsel/employee-relations coordination before mechanics**, and the packet carries the counsel-review banner when `legal.counsel_review: true`. Work items in the per-departure folder (`offboarding/<employee-slug>/`); keep reasons and identifiable case detail in the confidential zone, not here.

> **Jurisdiction-specific items are flagged ⚖️ below.** Final-pay timing, accrued-leave payout, benefits-continuation eligibility/notice/window, required separation notices, and mass-layoff advance-notice obligations all depend on the work location and `company.size`. Scope them to `legal.jurisdictions`, name the jurisdiction, and **verify with counsel**. **Do not state a specific deadline, headcount threshold, notice period, or coverage duration as fact** — flag it for verification. Never fabricate one.

Owner key: **HR** = HR/People · **MGR** = manager · **PAY** = payroll/finance · **IT** = IT/security · **FAC** = facilities · **LEGAL** = counsel/legal-compliance · **ER** = employee-relations.

## Phase 0 — Type & risk gate (do this first)
| Done | Task | Owner |
|---|---|---|
| ☐ | Separation type confirmed: voluntary (resignation/retirement) · involuntary (termination) · layoff/RIF | HR |
| ☐ | Resignation documented in writing (voluntary) | HR |
| ☐ | Decision + documentation reviewed; performance/conduct/PIP history pulled (involuntary) — coordinate `employee-relations/` | ER / HR |
| ☐ | Counsel review of termination/layoff, at-will vs. notice, protected-class/retaliation risk ⚖️ | LEGAL |
| ☐ | Mass-layoff advance-notice obligation assessed against headcount / `company.size` and location ⚖️ *(do not assert a threshold — verify)* | LEGAL |
| ☐ | Layoff selection criteria documented and adverse-impact reviewed | ER / LEGAL |
| ☐ | Severance & release considered and drafted/reviewed by counsel ⚖️ | LEGAL |
| ☐ | Packet stamped with counsel-review banner when `legal.counsel_review: true` | HR |

## Phase 1 — Pre-departure (before the last day)
| Done | Task | Owner |
|---|---|---|
| ☐ | Last working day set and confirmed | HR / MGR |
| ☐ | Departure communication plan (team, stakeholders, clients) — private, respectful, consistent | MGR / HR |
| ☐ | Knowledge transfer plan: responsibilities, documentation, key contacts, in-flight work | MGR |
| ☐ | Reassignment / coverage of the departing person's work | MGR |
| ☐ | Access-revocation plan drafted and timed to the last-day plan (executed by a human, not automatically) | IT / HR |
| ☐ | Property-return method arranged (in-person or remote shipping kit) | FAC / IT |
| ☐ | Final-pay items prepared (see Phase 3) | PAY / HR |
| ☐ | Benefits-continuation notice prepared ⚖️ *(eligibility/notice/window jurisdiction-specific — verify)* | HR |
| ☐ | Exit-interview scheduled (voluntary only) | HR |

## Phase 2 — Last day
| Done | Task | Owner |
|---|---|---|
| ☐ | Separation conversation held (private, respectful; witness present for involuntary per policy) | MGR / HR |
| ☐ | Final-pay details communicated ⚖️ *(timing jurisdiction-specific — verify)* | HR / PAY |
| ☐ | Benefits-continuation information provided ⚖️ | HR |
| ☐ | Company property returned and logged (or return kit issued) | FAC / IT |
| ☐ | Access revoked per plan (SSO, email, systems, physical, third-party) — executed by IT/security | IT |
| ☐ | Final documents signed as applicable (acknowledgments; release only via counsel) ⚖️ | HR / LEGAL |
| ☐ | Exit interview conducted (voluntary only) | HR |
| ☐ | Personal effects returned to the employee | FAC |

## Phase 3 — Final pay & benefits (draft; verify jurisdiction)
| Done | Item | Owner |
|---|---|---|
| ☐ | Final paycheck prepared ⚖️ *(payout timing jurisdiction-specific — verify)* | PAY |
| ☐ | Accrued-but-unused leave payout if owed ⚖️ *(whether owed is jurisdiction/policy-specific — verify)* | PAY / HR |
| ☐ | Outstanding expense reimbursements settled | PAY |
| ☐ | Authorized deductions / repayments reconciled ⚖️ *(what's permissible is jurisdiction-specific — verify)* | PAY |
| ☐ | Severance paid per counsel-approved terms, if any ⚖️ | PAY / LEGAL |
| ☐ | Benefits end date and continuation (COBRA-type) option communicated ⚖️ *(window/notice jurisdiction-specific — verify)* | HR |
| ☐ | Retirement/pension account handling explained | HR |

## Phase 4 — Post-departure
| Done | Task | Owner |
|---|---|---|
| ☐ | Access-revocation completeness verified (no lingering accounts/keys/badges) | IT / FAC |
| ☐ | Employee record updated and moved to the confidential zone; retention rules applied ⚖️ | HR |
| ☐ | HRIS status updated by a human (never automatically) | HR |
| ☐ | Final documents filed (confidential) | HR |
| ☐ | Exit-interview themes summarized into `reports/analytics/` — aggregated/anonymized, no individual PII exposed | HR |
| ☐ | Team continuity confirmed; backfill decision routed to recruitment if needed | MGR / HR |
| ☐ | Alumni/reference/rehire-eligibility policy noted | HR |
